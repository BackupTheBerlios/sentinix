Security

WARNING! The applications found under this interface may be used to perform Denial Of Service attacks against this host and/or other hosts! To guard against any such attacks make sure that this interface is only accessible from origins (and by people) you trust! It is recommended that you enable the firewall capabilities (Netfilter) on this host.

Edit /etc/rc.d/rc.firewall on this server and add your trusted networks. Login as root and run... 

$ cd /etc/rc.d
$ nano rc.firewall
$ chmod +x rc.firewall
$ ./rc.firewall start

As long as rc.firewall is chmodded +x, the firewall will be enabled after a reboot.

You may also want to restrict access to certain areas in this interface. By default, this GUI is only accessible through SSLv3 client authentication, most conveniently accomplished using a PKCS#12 (Public-Key Cryptography Standards) file (sentinix_client.p12). This key is automatically generated the first time SENTINIX boots (or if the server certificates are removed). In order to change these restriction policies, edit /etc/apache/sentinix.conf and read the comments.

SENTINIX is powered by these Open Source projects:
openMosix™ - a Linux kernel extension for single-system image clustering.
Nagios® - a very advanced host and service monitor.
Snort™ - a Network Intrusion Detection System.
SnortCenter™ - a web-based client-server management system for Snort.  
ACID™ - an analysis engine to search and process a database of security events generated by various IDSes, firewalls, and network monitoring tools.  
Nessus® - a remote security scanner (auditing).
Cacti™ - a network monitoring/graphing frontend to RRDTool.
RRDTool™ - you can think of RRDtool as a reimplementation of MRTGs graphing and logging features.
BandwidthD™ - tracks usage of TCP/IP network subnets and builds html files with graphs to display utilization.
Postfix™ - the MTA of choice for SENTINIX.
MailScanner™ - the eminent Anti-Virus and Anti-Spam Filter.
SpamAssassin™ - the eminent identifier of spam.
ClamAV™ - Clam AntiVirus is a GPL anti-virus toolkit for UNIX. The main purpose of this software is the integration with mail servers (attachment scanning).
MySQL™ - a fast SQL database.
