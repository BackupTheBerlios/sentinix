Field Variables

You can use variables in certain fields throughout Cacti and their values will be automatically substituted on the fly. The only two fields that currently support this are 'Title' for graphs and 'Name' for data sources. Below is a list of variables that can be used in these fields.

Host Fields
Variable Name Friendly Field Name
|host_description| Hostname
|host_hostname| Description
|host_management_ip| Management IP
|host_snmp_community| SNMP Community
|host_snmp_version| SNMP Version
|host_snmp_username| SNMP Username
|host_snmp_password| SNMP Password
