Suggested Values

Suggested values allow you to "suggest" default values to the user for fields that would otherwise not have a value. For each suggested value in Cacti, you must enter a field name and corresponding value. Below is a list of field names that can be used when creating suggested values:

Graph Fields
Cacti Field Name Friendly Field Name
title Title
image_format_id Image Format
height Height
width Width
auto_scale Auto Scale
auto_scale_opts Auto Scale Options
auto_scale_log Logarithmic Auto Scaling
auto_scale_rigid Rigid Boundaries Mode
auto_padding Auto Padding
export Allow Graph Export
upper_limit Upper Limit
lower_limit Lower Limit
base_value Base Value
unit_value Unit Value
unit_exponent_value Unit Exponent Value
vertical_label Vertical Label
