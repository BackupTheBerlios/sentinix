SENTINIX NEED MIRRORS

Want to provide a mirror? That would be very appreciated! I have now started rsync on sentinix.org. I can no longer maintain mirrors! A crontabbed daily (at most) rsync job on your side is the best way to do it. You can also mirror it using wget (or similar), put that's not recommended since it'll allocate precious FTP connections. There's, however, also a limitation on the number of concurrent rsync connections.

This will grab the sentinix-release module which contains full ISO releases. 

rsync -vr --delete rsync://sentinix.org/sentinix-release /where/you/want/it/

To do it with a wget command (not recommended) do this: 

$ cd /where/you/want/it/
$ wget -r -nH --cut-dirs=2 ftp://sentinix.org/pub/sentinix-release
